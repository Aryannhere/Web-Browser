# 🌐 Mini Web Browser

A simple web browser application built using **Node.js, Express.js, HTML, CSS, and JavaScript**.
This project simulates a basic browser interface with navigation controls and search functionality.

---

## 🚀 Features

* 🔍 Search or enter website URL
* 🌐 Load webpages inside the app
* ⬅ Back and ➡ Forward navigation
* 🔄 Reload page functionality
* 🎨 Chrome-like user interface

---

## 🛠 Technologies Used

* Node.js
* Express.js
* HTML
* CSS
* JavaScript

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/my-web-browser.git
```

2. Navigate to the project folder:

```
cd my-web-browser
```

3. Install dependencies:

```
npm install
```

4. Start the server:

```
node server.js
```

5. Open your browser and go to:

```
http://localhost:3000
```
